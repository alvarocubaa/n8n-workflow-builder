# Marketing Department — Verified Workflow Examples

Workflow JSONs generated by the n8n Workflow Builder and validated via `tools/audit_workflow.py`.
All workflows use Marketing department credentials from `chat-ui/src/lib/departments.ts`.

## How These Were Created

1. Each workflow was generated by the AI chat UI using the Marketing department context
2. Validated with `python3 tools/audit_workflow.py <file> --expected-creds '...'`
3. Deployed to personal n8n project for manual verification
4. Test cases in `tools/test_cases.yaml` can reproduce them
