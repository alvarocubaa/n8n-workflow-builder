Root cause & fix details ▼

Root cause: Every keystroke triggered a React re-render of ALL message bubbles. Each bubble contains heavy Markdown + Prism syntax highlighting — in a 30+ message conversation, that's 30+ expensive re-renders per character typed.

3 fixes applied:

Fix 1 React.memo on MessageBubble — skips re-render unless content actually changed
Fix 2 Memoized MessageList wrapper — isolates input state from message rendering tree
Fix 3 Stable message keys via crypto.randomUUID() — proper React reconciliation
